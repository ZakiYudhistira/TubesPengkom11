Source Code menggunakan banyak library Python tambahan dan diperlukan instalasi manual.
Untuk instalasi, jalankan code berikut pada Command Prompt atau Terminal

1. cProfile
pip install cProfile

2. datetime
Library untuk mengolah waktu
pip install datetime

3. pyrebase
Library untuk mengkoneksikan dan menggunakan fungsi Firebase dari Google untuk autentikasi dan database
pip install pyrebase4

4. PIL
Library untuk mengolah gambar menggunakan Python
pip install Pillow

5. openpyxl
Library untuk menghubungkan Python dengan Excel
pip install openpyxl

6. tkinter
Library bawaan dari Python, tidak perlu install
